# Пример микросервисов

## Как развернуть
1. Запустите `docker compose up -d --build`.
2. Запустите скрипт `entrypoint.sh`.

Скрипт `entrypoint.sh` для каждого микросервиса:
- Создаст .env файл
- Установит необходимые пакеты
- Создаст симлинк
- Установит необходимые права на папки
- Запустит миграции
